MAJOR PROJECTS

MUSIC TRACKER APP

GITHUB

The main purpose of this project is to aid in tracking and rating albums you have listened to. However, there are plenty of other functions that naturally coincide with this purpose, such as an account system, creating lists (Top 5 Artists, Albums, etc) for each account, adding new albums and artists into the database, and more were needed for the basic functionality of the project.

LANGUAGES / SKILLS USED:

  • React
  • JavaScript
  • MySQL
  • Node.js
  • Express

WORKOUT TRACKER APP

GITHUB

The Workout Tracker is for tracking one's progress in the gym, which is done by tracking how many sets, reps, and how much weight was done for each specific workout. This application is meant to be simple, because it is meant to be used while in the gym. Currently, only the frontend is deployed, so signing in and fetching/uploading user data does not work.

LANGUAGES / SKILLS USED:

  • HTML
  • TailwindCSS
  • JavaScript
  • MongoDB
  • Node.js
  • Express
